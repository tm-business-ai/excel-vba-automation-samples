# ExcelVBA業務自動化サンプル集

## 概要

日々の事務処理や社内業務で発生しやすい、転記、集計、入力チェックをExcelVBAで自動化するサンプル集です。

Excelだけで完結しやすい業務を想定し、CSVデータの読込、帳票への転記、集計表の作成、入力エラーの検出など、実務で使う場面が多い処理を中心にまとめています。

掲載しているデータ、会社名、担当者名、商品名、管理番号などはすべて架空のものです。

## このサンプル集で解決できる課題

- 手作業による転記ミスを減らしたい
- 毎月同じ形で作成する集計表を効率化したい
- 入力漏れ、形式不備、重複データを早めに検出したい
- Excel作業の属人化を減らし、手順を標準化したい
- CSVデータをExcel帳票や確認用シートへ取り込みたい

## 含まれるサンプル

| フォルダ | 内容 |
| --- | --- |
| `01_data_transfer_automation` | 入力データを指定フォーマットの帳票へ自動転記するサンプル |
| `02_monthly_summary_macro` | 売上CSVを読み込み、月別、商品別、担当者別に集計するサンプル |
| `03_input_validation_checker` | 未入力、数値エラー、日付形式エラー、重複データをチェックするサンプル |

## 使用技術

- Microsoft Excel
- Excel VBA
- CSV
- 標準モジュール
- ワークシート操作
- Dictionaryによる集計、重複チェック

## 使い方

1. 各サンプルフォルダのREADMEを確認します。
2. `sample_data`内のCSVをExcelに取り込みます。
3. `vba`内の`.bas`ファイルをExcel VBAエディタへインポートします。
4. READMEに記載された想定シート名に合わせてシートを作成します。
5. マクロを実行し、処理結果を確認します。

Excelファイル自体は含めていません。実際に使用する場合は、ローカル環境でマクロ有効ブックを作成してください。

## 各サンプルの概要

### 01_data_transfer_automation

入力シートのデータを確認し、帳票シートの指定セルへ転記します。入力チェック、帳票クリア、転記完了メッセージを含めた基本的な帳票作成処理を想定しています。

### 02_monthly_summary_macro

売上CSVをExcelへ取り込み、売上日、商品名、担当者をもとに集計表を作成します。月別、商品別、担当者別の3種類の集計結果を出力する想定です。

### 03_input_validation_checker

入力データの未入力、数値不備、日付形式不備、重複管理番号をチェックします。エラー一覧シートへの出力と、該当セルの色付けを行う想定です。

## GitHubに含めていないもの

- 実際の業務で使用したExcelファイル
- 顧客情報、取引先情報、個人情報を含むデータ
- 社内帳票や社内フォーマットそのもの
- マクロ有効ブック
- 環境変数ファイル
- ログ、バックアップ、一時ファイル

## セキュリティ上の注意

- このリポジトリには実データを含めない前提です。
- サンプルCSVはすべて架空データです。
- 実運用へ流用する場合は、個人情報、取引情報、社内ルールに応じて保存先、アクセス権、ログ出力内容を確認してください。
- Excelファイルを公開する場合は、非表示シート、名前定義、コメント、ドキュメント情報に不要な情報が残っていないか確認してください。

## 開発で意識したこと

- 実務で起こりやすい定型作業を題材にすること
- 処理の流れがGitHub上でも読みやすいこと
- 入力チェックやエラー処理を省略しないこと
- シート名、列構成、処理単位をREADMEに明記すること
- 実データや機密情報を含めないこと
