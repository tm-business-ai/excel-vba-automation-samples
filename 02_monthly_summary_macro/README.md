# 月次集計マクロ

## 概要

売上CSVを読み込み、月別、商品別、担当者別の集計表を自動作成するExcelVBAサンプルです。

毎月同じ手順で行う売上集計、営業実績確認、簡易レポート作成を想定しています。

## 解決できる課題

- CSVを手作業で並べ替え、集計する時間を減らす
- 集計条件の違いによる集計ミスを防ぐ
- 月別、商品別、担当者別の結果を同じ処理で出力する
- 集計表の作成手順を標準化する

## 主な機能

- CSVファイルの選択と取込
- 取込データの基本チェック
- 月別売上集計
- 商品別売上集計
- 担当者別売上集計
- 集計結果シートの作成、初期化
- エラー発生時のメッセージ表示

## 想定シート構成

| シート名 | 用途 |
| --- | --- |
| `売上データ` | CSVから取り込んだ明細データ |
| `月別集計` | 売上月ごとの集計結果 |
| `商品別集計` | 商品名ごとの集計結果 |
| `担当者別集計` | 担当者ごとの集計結果 |

## サンプルデータの説明

`sample_data/sales_sample.csv`には、架空の売上明細を入れています。

主な列は次の通りです。

- `売上日`
- `伝票番号`
- `商品コード`
- `商品名`
- `担当者`
- `数量`
- `単価`
- `売上金額`

## VBAコードの説明

`vba/MonthlySummary.bas`に標準モジュールとして利用できるコードを配置しています。

- `ImportSalesCsv`: CSVを選択して売上データシートへ取り込みます。
- `CreateMonthlySummary`: 月別、商品別、担当者別の集計表を作成します。
- `CreateSummaryByKey`: 指定列をキーにして売上金額を集計します。
- `PrepareSheet`: 出力用シートを作成または初期化します。

## 使い方

1. Excelで新規ブックを作成します。
2. VBAエディタで`vba/MonthlySummary.bas`をインポートします。
3. `ImportSalesCsv`を実行し、`sample_data/sales_sample.csv`を選択します。
4. `CreateMonthlySummary`を実行します。
5. 各集計シートの結果を確認します。

## セキュリティ上の注意

- サンプルデータはすべて架空データです。
- 実運用では、売上データや担当者情報の取扱ルールに従ってください。
- 外部から受け取ったCSVを扱う場合は、内容と保存場所を確認してから取り込んでください。
