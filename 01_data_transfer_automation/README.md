# 転記自動化ツール

## 概要

入力シートに貼り付けたデータを、指定フォーマットの帳票シートへ自動転記するExcelVBAサンプルです。

申請書、依頼書、社内管理表など、毎回同じ位置へ情報を転記する業務を想定しています。

## 解決できる課題

- 手入力やコピー貼り付けによる転記ミスを減らす
- 帳票作成にかかる時間を短縮する
- 入力漏れを転記前に確認する
- 帳票の初期化と作成手順を標準化する

## 主な機能

- 入力シートの必須項目チェック
- 帳票シートの既存内容クリア
- 入力内容の指定セルへの転記
- 明細行の繰り返し転記
- 処理結果メッセージの表示

## 想定シート構成

| シート名 | 用途 |
| --- | --- |
| `入力` | CSVや手入力で転記元データを配置するシート |
| `帳票` | 指定フォーマットへ転記結果を出力するシート |

入力シートは、1行目を見出し、2行目以降を明細データとして扱います。

## サンプルデータの説明

`sample_data/input_sample.csv`には、架空の依頼データを入れています。

主な列は次の通りです。

- `依頼番号`
- `依頼日`
- `部署`
- `担当者`
- `品目`
- `数量`
- `希望納期`
- `備考`

## VBAコードの説明

`vba/DataTransfer.bas`に標準モジュールとして利用できるコードを配置しています。

- `TransferDataToForm`: 入力チェック後、帳票へ転記します。
- `ClearForm`: 帳票シートの出力範囲をクリアします。
- `ValidateInputData`: 必須項目と数量、日付を確認します。
- `GetLastRow`: 指定列の最終行を取得します。

## 使い方

1. Excelで新規ブックを作成します。
2. `入力`、`帳票`の2シートを作成します。
3. `sample_data/input_sample.csv`の内容を`入力`シートへ貼り付けます。
4. VBAエディタで`vba/DataTransfer.bas`をインポートします。
5. `TransferDataToForm`を実行します。
6. `帳票`シートの転記結果を確認します。

## セキュリティ上の注意

- サンプルデータはすべて架空データです。
- 実運用では、帳票内に個人情報や取引先情報が残ったまま保存されないよう注意してください。
- 公開前に非表示シート、コメント、ドキュメント情報も確認してください。
