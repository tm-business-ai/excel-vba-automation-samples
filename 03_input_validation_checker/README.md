# 入力チェック・エラー検出ツール

## 概要

入力データの未入力、数値エラー、日付形式エラー、重複データを自動チェックするExcelVBAサンプルです。

申込一覧、管理台帳、CSV取込後の確認作業など、入力品質を確認してから次工程へ進める業務を想定しています。

## 解決できる課題

- 入力漏れや形式不備の見落としを減らす
- 重複データを早めに検出する
- エラー箇所を目視で確認しやすくする
- チェック結果を一覧化し、修正作業を進めやすくする

## 主な機能

- 必須項目の未入力チェック
- 数値項目の形式チェック
- 日付項目の形式チェック
- 管理番号の重複チェック
- エラー一覧シートへの出力
- エラー該当セルの色付け
- チェック結果メッセージの表示

## 想定シート構成

| シート名 | 用途 |
| --- | --- |
| `入力データ` | チェック対象データを配置するシート |
| `エラー一覧` | 検出したエラー内容を出力するシート |

入力データシートは、1行目を見出し、2行目以降をチェック対象として扱います。

## サンプルデータの説明

`sample_data/input_check_sample.csv`には、チェック動作を確認しやすいように、あえて一部に入力不備を含めた架空データを入れています。

主な列は次の通りです。

- `管理番号`
- `受付日`
- `部署`
- `担当者`
- `金額`
- `数量`
- `ステータス`

## VBAコードの説明

`vba/InputValidationChecker.bas`に標準モジュールとして利用できるコードを配置しています。

- `CheckInputData`: 入力データ全体をチェックします。
- `AddError`: エラー一覧へ内容を追加し、該当セルに色を付けます。
- `ClearCheckResult`: 前回のチェック結果をクリアします。
- `PrepareErrorSheet`: エラー一覧シートを作成または初期化します。

## 使い方

1. Excelで新規ブックを作成します。
2. `入力データ`シートを作成します。
3. `sample_data/input_check_sample.csv`の内容を`入力データ`シートへ貼り付けます。
4. VBAエディタで`vba/InputValidationChecker.bas`をインポートします。
5. `CheckInputData`を実行します。
6. `エラー一覧`シートと、色付けされたセルを確認します。

## セキュリティ上の注意

- サンプルデータはすべて架空データです。
- 実運用では、エラー一覧に個人情報や機密情報をそのまま出力しないよう注意してください。
- 入力元ファイルを公開する場合は、不要な列や非表示情報が残っていないか確認してください。
